Copy Button

Bağımsız kopyala butonu. <c-code-block> içindeki kopyala butonunun standalone versiyonu. Herhangi bir metin veya element içeriğini panoya kopyalar.

Doğrudan Metin

text attribute'u ile kopyalanacak metin doğrudan verilir. Butonun sağındaki örnekte tıklayın:

npm install legacy-doc

Element Hedefleme

target attribute'una CSS seçici verin; o elementin textContent'i kopyalanır. 

const greet = name => `Merhaba, ${name}!`;

Özel Etiket

label attribute'u ile buton metnini değiştirebilirsiniz:

Kod

<!-- Doğrudan metin --> <c-copy-button text="npm install legacy-doc"></c-copy-button> <!-- Element hedefleme --> <code id="my-snippet">const x = 42;</code> <c-copy-button target="#my-snippet" label="Kodu Kopyala"></c-copy-button>
