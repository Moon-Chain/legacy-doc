Banner

Sayfa üstü duyuru çubuğu. Deprecation uyarıları, beta etiketleri, kırıcı değişiklik bildirimleri veya genel duyurular için kullanılır. Kapatılabilir (dismissible) ve kalıcı kapatma hafızası (persist-key) destekler.

Varyantlar

Kapatılabilir (Dismissible)

dismissible attribute'u eklenince sağ köşede × butonu çıkar. persist-key ile kapanma durumu localStorage'a kaydedilir; sayfa yenilenince de kapalı kalır.

HTML İçerik

Link veya <strong> gibi inline HTML, message yerine slot içeriği olarak da kullanılabilir.

Kırıcı Değişiklik: v-model API'si değişti. Migration rehberini incele →

Kod

<!-- Temel kullanım --> <c-banner variant="info" message="Genel duyuru metni."></c-banner> <!-- Kapatılabilir --> <c-banner variant="warning" dismissible message="Bu API kaldırılacak."></c-banner> <!-- Kalıcı kapatma hafızası --> <c-banner variant="beta" dismissible persist-key="banner-v1" message="Beta aşamasındadır."></c-banner> <!-- HTML içerik (slot) --> <c-banner variant="danger" dismissible> <strong>Kırıcı Değişiklik:</strong> <a href="/migration">Rehber &rarr;</a> </c-banner>

Attribute'lar

AttributeTipVarsayılanAçıklama
variant string info info | warning | danger | success | beta
message string — Gösterilecek metin. Verilmezse element içeriği (slot) kullanılır; HTML destekler.
dismissible boolean false Varlığı yeterli. Sağ köşede × butonu gösterir.
persist-key string — localStorage key'i. Verilirse kapatma durumu kaydedilir; sayfa yenilenince banner tekrar gösterilmez.
