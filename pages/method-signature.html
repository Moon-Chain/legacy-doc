Method Signature

TypeScript stili fonksiyon/metod imzası gösterimi. Fonksiyon adı, parametre adları/tipleri ve dönüş tipi renklendirilir. API referans sayfalarında <c-api-endpoint> ile iyi bir ikili oluşturur.

Structured Mod

name ve returns attribute'ları ile <c-param> elementleri kullanılır. optional attribute'u olan parametrelerin adına otomatik ? eklenir.

Düz Metin Modu

name attribute'u verilmezse içerik olduğu gibi mono fontla render edilir. Mevcut imza stringlerini hızlıca göstermek için kullanışlıdır.

render(template: string, data?: object): string dispose(): void

Kod

<!-- Structured mod --> <c-method-signature name="getUserById" returns="Promise<User>"> <c-param name="id" type="string"></c-param> <c-param name="options" type="RequestOptions" optional></c-param> </c-method-signature> <!-- Düz metin modu --> <c-method-signature>render(template: string): string</c-method-signature>
