Search Box

İstemci tarafında (client-side) filtreleme yapan arama kutusu. target seçicisine uyan elementlerden, girilen kelimeyi metninde içermeyenler gizlenir. Ana sayfadaki bileşen galerisi araması, <c-search-box>'ın canlı bir örneğidir.

Örnek

  • GET /api/users
  • POST /api/users
  • PUT /api/users/{id}
  • PATCH /api/users/{id}
  • DELETE /api/users/{id}
  • GET /api/posts
  • POST /api/posts

Kullanım

<c-search-box target=".c-demo-list li" placeholder="Bileşen ara..." empty-text="Eşleşen sonuç bulunamadı."> </c-search-box> <ul class="c-demo-list"> <li>...</li> <li>...</li> </ul> Kullanıcı yazdıkça, target seçicisine uyan her element için textContent küçük harfe çevrilip arama kelimesiyle karşılaştırılır. Eşleşmeyenler display: none ile gizlenir. Hiçbir eşleşme yoksa empty-text mesajı gösterilir.

Attribute'lar

Attribute Açıklama Varsayılan
target Filtrelenecek elementlerin CSS seçicisi —
placeholder Input placeholder metni Ara...
empty-text Hiç sonuç bulunamadığında gösterilecek mesaj Sonuç bulunamadı.
