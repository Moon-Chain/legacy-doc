Code Block

Sözdizimi vurgulamalı (syntax highlight) ve kopyala butonlu kod bloğu. Vurgulama için Prism.js kullanılır.

Örnek

function topla(a, b) { return a + b; } console.log(topla(2, 3));

Kullanım

<c-code-block lang="javascript"> // kod buraya </c-code-block> Bileşenin içine yazılan kod, textContent olarak okunur ve <pre><code> içine yerleştirilir. Bu nedenle < ve > karakterlerini &lt; / &gt; olarak yazmaya gerek yoktur — HTML olarak değil düz metin olarak işlenir.

Diğer diller

.c-card { display: flex; gap: var(--c-space-3); border: 1px solid var(--c-border); border-radius: var(--c-radius); background: var(--c-bg-elevated); } { "id": 42, "name": "Ada Lovelace", "role": "admin", "active": true, "score": 98.6, "tags": ["math", "computing"], "meta": null } <div class="c-card"> <p>Merhaba</p> </div>

Attribute'lar

Attribute Açıklama Varsayılan
lang Prism dil sınıfı (language-{lang}). Örn: javascript, css, markup, json, php. markup

Kopyala butonu

Sağ üst köşedeki kopyala butonu, fare ile üzerine gelindiğinde görünür ve navigator.clipboard API'sini kullanarak kod içeriğini panoya kopyalar. Kopyalama sonrası buton kısa süreliğine "Kopyalandı" metnine döner.
