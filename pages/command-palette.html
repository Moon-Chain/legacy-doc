Command Palette

Ctrl+K (Mac'te ⌘+K) kısayoluyla açılan klavye-driven bileşen arama overlay'i. Tüm bileşen sayfalarını anlık filtreler; ok tuşlarıyla gezilir, Enter ile hedef sayfaya yönlendirir.

Deneme

Aşağıdaki butona tıkla veya Ctrl+K kısayolunu kullan.

Özellikler

  • Tüm bileşen sayfalarını arar (49 sayfa)
  • Yazarken anlık filtreler — başlık ve grup adına göre
  • ↑ / ↓ ok tuşlarıyla gezinme
  • Enter ile seçili sayfaya git
  • Dışarı tıklama veya Esc ile kapat
  • Açık/koyu tema desteği

Kullanım

<c-command-palette> elementini sayfanın herhangi bir yerine ekle. Script yüklendiğinde global Ctrl+K kısayolunu otomatik dinlemeye başlar.

<!-- CSS --> <link rel="stylesheet" href="components/c-command-palette/c-command-palette.css"> <!-- Element (body'nin herhangi bir yerine) --> <c-command-palette base="./"></c-command-palette> <!-- Script --> <script src="components/c-command-palette/c-command-palette.js"></script> <!-- Programatik açma --> <script> document.querySelector('c-command-palette').open(); </script>

Attribute'lar

AttributeTipVarsayılanAçıklama
