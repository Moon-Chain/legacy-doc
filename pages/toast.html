Toast

Geçici bildirim mesajları. Sağ alt köşede yığılarak görünür, belirli süre sonra otomatik kapanır. Statik CToast.show() yöntemiyle JS'den tetiklenir.

Varyantlar

Kalıcı Toast

duration: 0 ayarlanırsa toast otomatik kapanmaz; kullanıcı × butonuna basmalıdır.

Kullanım

// Temel kullanım CToast.show('Kaydedildi!'); // Varyant ve süre CToast.show('Hata oluştu.', { variant: 'danger', duration: 5000 }); // Kalıcı (kullanıcı kapatana kadar açık) CToast.show('Güncelleme mevcut.', { variant: 'info', duration: 0 }); <!-- HTML element olarak kullanmak için --> <c-toast id="t" variant="success" message="Kaydedildi!"></c-toast> // Element formu document.getElementById('t').show();

Attribute'lar (element formu)

AttributeTipVarsayılanAçıklama
variantstringinfoinfo | success | warning | danger
messagestring—Gösterilecek metin (ya da element içeriği)
durationnumber3500ms cinsinden süre; 0 = kalıcı
