Steps

Numaralı adım rehberi. "3 adımda kurulum" gibi sıralı süreçleri görsel olarak düzenlemek için kullanılır.

Kurulum Örneği

Projeyi yerel ortamına indirmek için:

git clone https://github.com/example/legacy-doc.git

Proje kök dizininde aşağıdaki komutu çalıştır:

npm install

Uygulamayı yerel olarak çalıştır:

npm start

Tarayıcıda http://localhost:3000 adresine git.

İstediğin <c-*> elementlerini sayfana ekle — sunucu gerekmez, file:// uyumludur.
