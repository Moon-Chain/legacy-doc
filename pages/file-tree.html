File Tree

Açılır/kapanır klasör ağacı. İç içe geçmiş klasör yapılarını görsel olarak göstermek için kullanılır. Klasöre tıklayarak aç/kapat.

Proje Yapısı Örneği

Kod

<c-file-tree> <c-tree-folder name="src" open> <c-tree-folder name="components"> <c-tree-file name="Button.tsx"></c-tree-file> <c-tree-file name="Input.tsx"></c-tree-file> </c-tree-folder> <c-tree-file name="index.ts"></c-tree-file> </c-tree-folder> <c-tree-file name="package.json"></c-tree-file> <c-tree-file name="tsconfig.json"></c-tree-file> </c-file-tree>

Attribute'lar

ElementAttributeAçıklama
c-tree-foldernameKlasör adı
c-tree-folderopenBaşlangıçta açık göster (boolean)
c-tree-filenameDosya adı. Uzantıya göre ikon otomatik seçilir.
