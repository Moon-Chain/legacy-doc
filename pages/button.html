Button

Sadece CSS bileşeni. .btn sınıfı herhangi bir <button>, <a> veya <input type="button"> üzerinde kullanılabilir.

Varyantlar

Default .btn
Primary .btn--primary
Secondary .btn--secondary
Success .btn--success
Danger .btn--danger
Ghost .btn--ghost

Outline

Outline Primary .btn--outline.btn--primary
Outline Secondary .btn--outline.btn--secondary
Outline Success .btn--outline.btn--success
Outline Danger .btn--outline.btn--danger

Boyutlar

Small .btn--sm
Medium (varsayılan) .btn
Large .btn--lg

Devre Dışı

Default Disabled disabled
Primary Disabled disabled
Danger Disabled disabled

Kod

<!-- Temel --> <button class="btn">Varsayılan</button> <!-- Varyantlar --> <button class="btn btn--primary">Primary</button> <button class="btn btn--danger">Danger</button> <button class="btn btn--ghost">Ghost</button> <!-- Outline --> <button class="btn btn--outline btn--primary">Outline Primary</button> <!-- Boyutlar --> <button class="btn btn--primary btn--sm">Küçük</button> <button class="btn btn--primary btn--lg">Büyük</button> <!-- Link olarak --> <a href="/..." class="btn btn--primary">Link</a> <!-- Devre dışı --> <button class="btn btn--primary" disabled>Devre Dışı</button>
