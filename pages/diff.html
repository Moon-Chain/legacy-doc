İki kod parçasını yan yana karşılaştıran satır bazlı fark görüntüleyici. Eklenen ve silinen satırlar vurgulanır; bitişik değişiklikte kelime düzeyinde inline fark gösterilir. Prism.js ile sözdizimi renklendirmesi desteklenir.

<c-diff lang="javascript" label-from="v1.js" label-to="v2.js"> <script type="text/plain" slot="from"> // eski kod buraya </script> <script type="text/plain" slot="to"> // yeni kod buraya </script> </c-diff>

Attribute'lar

AttributeTipVarsayılanAçıklama
langstringmarkupPrism dil tanımlayıcısı (javascript, json, css, typescript vb.)
label-fromstringbeforeSol etiket (ör. dosya adı veya sürüm)
label-tostringafterSağ etiket

Slot'lar

SlotAçıklama
fromEski / kaynak kod. <script type="text/plain"> olarak verilmelidir.
toYeni / hedef kod. <script type="text/plain"> olarak verilmelidir.
