Prop Table

Bileşen attribute/prop referans tablosu. JSON dizisi olarak verilen prop tanımlarını otomatik olarak stillendirilmiş bir tabloya dönüştürür. Her sayfadaki elle yazılmış HTML tablolarının yerini alır.

Temel Kullanım

props attribute'una JSON dizisi verin; her eleman name, type, default, required ve description alanlarını içerir.

Kod

<c-prop-table props='[ {"name":"variant","type":"string","default":"info","required":false,"description":"Renk teması"}, {"name":"dismissible","type":"boolean","default":"—","required":false,"description":"Kapatma butonu"}, {"name":"title","type":"string","default":"—","required":false,"description":"Başlık metni"} ]'></c-prop-table>

Zorunlu Prop Örneği

required: true olan satırlar kırmızıyla vurgulanır.
