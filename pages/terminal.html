Terminal

Sahte terminal penceresi. Prompt + komut + çıktı satırlarını satır türüne göre renklendirir. c-code-block'tan farkı: etkileşimli bir terminal oturumunu görsel olarak simüle eder.

Kurulum Örneği

# Repoyu klonla git clone https://github.com/example/legacy-doc.git Cloning into 'legacy-doc'... remote: Enumerating objects: 214, done. cd legacy-doc && npm install added 42 packages in 1.4s npm start Server running on http://localhost:3000

Hata Örneği

npm install --global some-package npm ERR! Error: EACCES: permission denied, mkdir '/usr/local/lib/node_modules' npm ERR! Please try running this command again as root/Administrator. # sudo ile tekrar dene sudo npm install --global some-package added 1 package in 0.8s

Kod

<c-terminal title="Kurulum" prompt="$"> <c-terminal-line type="comment"># Repoyu klonla</c-terminal-line> <c-terminal-line type="input">git clone https://github.com/example/repo.git</c-terminal-line> <c-terminal-line type="output">Cloning into 'repo'...</c-terminal-line> <c-terminal-line type="error">Error: ENOENT: no such file or directory</c-terminal-line> </c-terminal>

Attribute'lar

ElementAttributeAçıklama
c-terminaltitleBaşlık çubuğu metni (varsayılan: bash)
c-terminalpromptInput satırlarındaki prompt sembolü (varsayılan: $)
c-terminal-linetypeinput | output | error | comment (varsayılan: düz beyaz)

Satır Türleri

typeGörünümKullanım
inputYeşil prompt + beyaz metinKullanıcının yazdığı komutlar
outputSoluk beyazKomut çıktısı, başarılı sonuçlar
errorKırmızıHata mesajları
commentGri italikAçıklama satırları (#...)
