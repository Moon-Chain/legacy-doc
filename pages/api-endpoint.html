API Endpoint

REST endpoint kartı. Method badge + URL + parametre tablosu + istek/yanıt örneklerini tek açılır/kapanır kart içinde gösterir. Swagger'a gerek kalmadan inline API dokümantasyonu yazmak için uygundur.

Örnek — Kullanıcı API

Kod

<c-api-endpoint method="POST" url="/api/v1/users" summary="Yeni kullanıcı oluşturur"> <script type="application/json"> { "params": [ { "name": "name", "type": "string", "required": true, "desc": "Kullanıcı adı" }, { "name": "email", "type": "string", "required": true, "desc": "E-posta adresi" }, { "name": "role", "type": "string", "required": false, "desc": "Kullanıcı rolü" } ], "request": "{ \"name\": \"Ali\", \"email\": \"ali@example.com\" }", "response": "{ \"id\": 1, \"name\": \"Ali\" }" } </script> </c-api-endpoint>

Attribute'lar

AttributeDeğerAçıklama
methodGET | POST | PUT | PATCH | DELETEHTTP metodu (varsayılan: GET)
urlstringEndpoint yolu, ör. /api/v1/users
summarystringKısa açıklama (opsiyonel)
