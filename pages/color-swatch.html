Color Swatch

CSS renk token'larını görselleştiren bileşen. tokens.css içindeki değişkenleri otomatik okuyarak hesaplanmış renk değerini, değişken adını ve etiketi bir arada gösterir. Tıklanınca değeri panoya kopyalar.

Temel Kullanım

Tema Renkleri

Aktif temaya göre değişen semantik token'lar. Karanlık modda geçiş yaparak farkı görebilirsin.

Block Varyantı

Renk bloğu üstte, bilgi altta. Büyük palet gösterimlerinde kullanılır.

Kod

<!-- Tek swatch --> <c-color-swatch var="--c-primary" label="Primary"></c-color-swatch> <!-- Block varyantı --> <c-color-swatch var="--c-success" label="Success" variant="block"></c-color-swatch> <!-- Yatay grid --> <div class="c-swatch-grid"> <c-color-swatch var="--c-primary" label="Primary"></c-color-swatch> <c-color-swatch var="--c-secondary" label="Secondary"></c-color-swatch> <c-color-swatch var="--c-success" label="Success"></c-color-swatch> </div> <!-- Blok palet --> <div class="c-swatch-palette"> <c-color-swatch var="--c-primary" label="Primary" variant="block"></c-color-swatch> <c-color-swatch var="--c-danger" label="Danger" variant="block"></c-color-swatch> </div>

Attribute'lar

AttributeTipVarsayılanAçıklama
var string — CSS değişken adı. --c-primary gibi tam adla verilmeli.
label string var değeri Gösterilecek başlık. Verilmezse değişken adı kullanılır.
variant string — block: renk bloğu üstte, bilgi altta büyük palet görünümü.
